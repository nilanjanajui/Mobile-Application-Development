# Mobile Application Development


This repository contains a collection of **Android applications** developed using **Android Studio**. Each folder represents a standalone Android project designed to demonstrate specific mobile application development concepts.

---

## Project Structure

```
.
├── bmi_calculator
├── celciustofarenheit
├── hello
├── loanpayment
└── saferoute
```

---

## Project Descriptions

### 1. BMI Calculator (`bmi_calculator`)

An Android application that calculates Body Mass Index (BMI) based on user-entered height and weight.

**Key Concepts:**

* User input using EditText
* Button click handling
* Mathematical calculations
* Displaying results using TextView

---

### 2. Celsius to Fahrenheit Converter (`celciustofarenheit`)

An Android app that converts temperature values from Celsius to Fahrenheit.

**Key Concepts:**

* Input validation
* Arithmetic conversion logic
* Interactive UI elements

---

### 3. Hello App (`hello`)

A basic Android application that displays a simple "Hello World" message.

**Key Concepts:**

* Android project setup
* Activity lifecycle basics
* XML layouts

---

### 4. Loan Payment Calculator (`loanpayment`)

An Android application that calculates loan payment amounts using principal, interest rate, and loan duration.

**Key Concepts:**

* Financial calculations
* Multiple input fields
* Structured result display

---

### 5. Safe Route App (`saferoute`)

An Android application concept focused on identifying safer travel routes.

**Key Concepts:**

* User-focused app design
* Location-based concepts
* Android permissions (if implemented)

---

## Technologies Used

* Android Studio
* Java
* XML
* Android SDK

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```
2. Open **Android Studio**.
3. Select **Open an Existing Project**.
4. Open any project folder from this repository.
5. Allow Gradle to sync.
6. Run the app on an emulator or physical device.
